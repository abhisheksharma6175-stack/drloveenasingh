# Dr. Loveena Singh MD Website

Modern medical website built with Next.js 15 App Router, Tailwind CSS, Framer Motion, React Icons, and `next/image`.

## Run Locally

```bash
npm.cmd install
npm.cmd run dev
```

Then open `http://localhost:3000`.

## Included

- Responsive sticky navbar with mobile menu
- Premium animated homepage sections
- About, services, gallery, blog, testimonials, and contact pages
- Dynamic blog routes from slug data
- Sitemap and robots routes
- Local generated doctor hero image in `public/images/loveena-singh.png`
- Remote optimized gallery/blog imagery via `next/image`
